# Fleet Tracking Dashboard

A real-time fleet monitoring dashboard built with React, Material-UI, and Leaflet. Visualizes 5 concurrent US trips with progress, alerts, and interactive maps (Uber-style navigation).

## Prerequisites
- Node.js (v18+): [Download here](https://nodejs.org)
- Git: Clone the repo (`git clone <your-repo-url>`)

## Setup
1. **Clone & Navigate**:
git clone <your-repo-url>
cd fleet-tracking-dashboard


2. **Install Dependencies**:
npm install

- Installs React, MUI, Leaflet, Moment, etc.

## Run Locally
1. **Start Development Server**:
npm start

- Opens http://localhost:3000 automatically.
- Hot-reloads on changes; Ctrl+C to stop.
